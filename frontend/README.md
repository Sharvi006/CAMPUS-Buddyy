

## Project info


**NAME**:campus-buddy-bot-70


**Project Description**
An AI powered (use of Gemini 2.5 flash) chatbot to solve real-world problems(self-based) for college related queries for students. It has been developed from static knowledge.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
